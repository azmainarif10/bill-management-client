 **Name : Bill Management System 
 **Live Url : https://bill-management-5eb98.web.app/

**App description : The Bill Management Website is a modern, user-friendly platform designed to help individuals and organizations efficiently manage their financial obligations.The system allows users to add, view, edit, and delete bills, each containing key details such as the bill title, category, amount, due date, location, description, and image. Through an intuitive interface, users can monitor upcoming payments, avoid late fees, and maintain a clear understanding of their financial responsibilities.

**Key features : 

 Add and Manage Bills: Create, update, or delete bills with detailed information.

 Track Due Dates: Easily view upcoming and past-due payments.
 User Authentication: Secure login and personalized dashboard for each user.
 Light/Dark Mode: User-friendly design with theme customization.




**Technologies used : HTML ,Tailwind Css , Javascript ,React
